
# FoodExplorer - Desafio Final

Back-End do desafio final da Trilha Explorer da RocketSeat. O FoodExplorer é um cardápio digital de um restaurante fictício. Desenvolvido utilizando NodeJS, KnexJS e ExpressJS.




## Funcionalidades
    
- Adicionar/Atualizar/Visualizar os pratos;
- Realizar um pedido;
- Verificar o status;
- Fazer o pagamento;
- Adicionar pratos favoritos;
- Verificar pedidos antigos;
- Acompanhar e atualizar o status dos pedidos (Admin).
